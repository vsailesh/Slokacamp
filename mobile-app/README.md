# SlokaCamp Mobile App

## Features
- User Authentication (Signup/Signin)
- Browse Courses
- Course Details
- Enroll in Courses
- Track Progress
- User Dashboard

## Tech Stack
- React Native (Expo)
- React Navigation
- Axios for API calls
- AsyncStorage for local storage

## Setup

```bash
cd mobile-app
yarn install
yarn start
```

## Run on Device

1. Install Expo Go app on your phone
2. Scan the QR code from terminal
3. App will open on your device

## Build for Production

```bash
# Android
eas build --platform android

# iOS
eas build --platform ios
```
